# Changelog

All notable changes to RSInfinity will be documented in this file.

## [2.0.0] - 2025-12-18

**Project Rebirth** - RSInfinity aims to find compatible ways to bring graphics enhancements back to Roblox after recent security updates blocked the original implementation.

### Changed
- Rebranded from Roshade to RSInfinity
- Updated manufacturer to Zaikonurami
- Renamed all internal directories and file references
- Established mission to collaborate with Roblox for official solutions

### Added
- Comprehensive documentation (README, ROADMAP, CONTRIBUTING, SECURITY)
- Historical context about Roblox security changes
- Project mission and collaboration roadmap
- Enhanced community guidelines

### Fixed
- File paths in configuration files
- Copyright information (2025)
- Installer output filename
